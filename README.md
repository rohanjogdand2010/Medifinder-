🏥 MediFinder

MediFinder is a modern healthcare web application that helps users find nearby medical shops and pharmacies using real-time geolocation and interactive maps.

✨ Features

- 📍 Auto Detect User Location
- 🗺️ OpenStreetMap + Leaflet Integration
- 🏥 Nearby Medical Shop Finder
- ⭐ Pharmacy Ratings & Reviews
- 📏 Distance Information
- 🚨 Emergency Pharmacy Support
- 🚚 Medicine Delivery Request Form
- 📞 Contact Form
- 📱 Fully Responsive Design
- 🎨 Modern Blue & White Healthcare Theme
- ⚡ Fast and Lightweight

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js
- OpenStreetMap
- Geolocation API

🚀 Getting Started

1. Clone the repository:

git clone https://github.com/your-username/MediFinder.git

2. Open the project folder.

3. Launch "index.html" in your browser.

4. Allow location access to find nearby pharmacies.
